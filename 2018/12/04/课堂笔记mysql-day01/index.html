6.MySQL常用的SQL命令（对数据进行操作）
SQL: Structured Query Language，结构化查询语言，用于操作关系型数据库服务器中的数据。——增删改查
SQL语言最早是由IBM提出，后提交给ISO，最终成为了数据库行业内的标准语言。分为多个版本:SQL-87、SQL-92、SQL-99。当前标准的SQL语言被绝大多数的关系型数据库所支持。
SQL命令的两种执行方式
(1)交互模式：客户端连接MySQL，输入一行，回车，服务器执行一行。适用于临时性的查看数据
mysql -uroot 回车
(2)脚本模式：客户端把要执行的多行命令写在一个脚本文件中，一次性的提交给服务器执行。适用于批量的操作数据进行增删改查。（不能登入mysql之后去执行）
mysql -uroot < C:/xampp/…../02.sql {此处是输入mysql -uroot < 然后把脚本文件拖拽过来按回车}
练习：创建02.sql脚本文件，在脚本中书写多个命令（显示所有数据库，进入PHPmyadmin，显示所有的数据表）提交给服务器执行，尝试写错其中一行，执行查看结果。
SQL语法规范
(1)每条SQL语句必须以英文分号作为结尾，一条语句可以跨越多行，见到分号认为结束。
(2)若第n条SQL语句有语法错误，则后续所有的语句不会再执行。
(3)SQL语句中不区分大小写。习惯上数据库关键字用大写，非关键字用小写。
(4)SQL语句还可以使用单行注释：#………. 或者多行注释: / … /，注释的内容不会被服务器所执行。
日常开发中常用的SQL命令

(1)丢弃数据库，如果存在的话
DROP DATABASE IF EXISTS jd;
(2)创建新的数据库
CREATE DATABASE jd;
(3)进入刚刚创建的数据库
USE jd;
(4)创建数据表
CREATE TABLE student(
sid INT,
name VARCHAR(6),
sex VARCHAR(1),
score INT
);
(5)向数据表中插入记录
INSERT INTO student VALUES
(‘10’,’Tom’,’M’,’89’)；
(‘11’,’Kate’,’F’,’92’)
INSERT INTO student VALUES(“1”,”tom”,”M”,”80”);
INSERT INTO student VALUES(“2”,”jerry”,”M”,”82”);
（6）删除学号为3的学生的数据（删）
DELETE FROM student WHERE id=”3”;
（7） 修改学号为2的学生的成绩为66（改）
UPDATE student SET score=”66”,sex=”F” WHERE id=”2”;
…..;
(6)查询数据表所有的记录
SELECT * FROM student;

标准SQL语句分类
DDL:DATA DEFINE LANGUAGE 定义数据结构
Create/drop/alter
DML:Data Munipulate Language 操作数据
INSERT/DELETE/UPDATE
DQL:Data Query Language 查询数据
SELECT
DCL:Data Control language 控制用户权限
GRANT(授权)、REVOKE(收权)
