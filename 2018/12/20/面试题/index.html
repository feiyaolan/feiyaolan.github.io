变量？
内存中存储数据的一片空间
变量的分类？
—全局变量
1，var在全局的变量 2，非严格模式下不带var的变量 eg: a=1;由程序自动声明一个变量放在当前作用域前
—局部变量 1，声明在局部作用域 2，函数的形参

赋值操作的返回值就是等号右边的内容

什么是返回值？return
返回值做几件事？
1， 返回函数调用的结果,return的就是它右边那一块，直到分号结束。
2， 终止当前函数的执行
什么时候需要return?
如果函数没有return，但是非要一个返回值，结果undefind

什么情况下产生undefined？
1， 声明了变量未赋值
2， 函数没有return, 程序默认返回undefined
3， 声明了一个函数有形参，但是不给形参赋值，默认undefined来补位，多了不接，按顺序读。（未指定实参由undefined按顺序补位）多余的实参会被忽略。
4， 访问数组不存在的项
5， 访问对象不存在的属性

区别：访问权限的问题—》局部只能在当前作用域或者当前作用域的子定义域，全局在哪儿都行。

前加加和后加加区别
Var a=1;
++a
a++
对变量而言：无区别，都会自增1
对表达式而言：前++：返回新值（+1后的值）
后++：返回旧值（+1之前的值）

值传递（复印）
函数的调用需要传入变量作为实参时:
1， 传递值—》针对原始类型而言，如string/number/boolean等，将原变量的值复制一份作为实参，今后对该实参做的所有操作与变量无关
2， 址传递—》针对引用类型而言，如函数function/object/array等

今日区分：全局变量，局部变量，return
//获取指定范围内的n个不重复的随机数
function fun(min,max,n){
var res=[];//存储结果
for (var i=0;i<n ;i++ )//外层负责生成随机数
{//获取指定范围（min~max之间）的随机数Math.random()(max-min+1)+min
var num=parseInt(Math.random()(max-min+1)+min);
res.push(num);
for (var j=0;j<i ;j++ ){//内层判断是否重复
if (num==res[j]){//拿循环当次获得的随机数与res现存的每一项比较，如果重复就删掉
res.pop(); //同时重复一遍随机，如果当前项已经重复就没必要继续判断
i–;
break;
}
}
}
return res;//返回结果
}
var arr=fun(1,5,5);
console.log(arr);
